# Switch

Act like native `checkbox`, switch allows you to toggle an option on and off.

<ComponentPreview
  client:load
  component='
    <breeze-switch>
      Remote only
    </breeze-switch>
  '
/>

```html
<breeze-switch>Remote only</breeze-switch>
```

<ComponentPreview
