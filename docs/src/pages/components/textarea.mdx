### Helper Text

  '
  style="display: block;"
/>

```html
<cc-textarea label="Message" helper-text="Please leave your message"></cc-textarea>
```

Or, using `slot` to add label and helper text if you need them to be HTML.

<ComponentPreview
  client:load
  component='
    <cc-textarea>

    </cc-textarea>
  '
  style="display: block;"
/>

```html
<cc-textarea>
  <p slot="label">Message</p>
  <p slot="helper-text">Please leave your message</p>
</cc-textarea>
```
