Vertical layout places components from top to bottom. By default, its flex children is set to `align-items: stretch;`.

<ComponentPreview
  client:load
  component='
    <cc-vertical-layout>
      <cc-button>Cancel</cc-button>
      <cc-button variant="primary">Save</cc-button>
    </cc-vertical-layout>
  '
  style="display: block;"
/>

```html
<cc-vertical-layout>
  <cc-button>Cancel</cc-button>
  <cc-button variant="primary">Save</cc-button>
</cc-vertical-layout>
```
